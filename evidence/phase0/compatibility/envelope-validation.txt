The SerializationEnvelope wraps any contract type with:
  - schema name and version
  - envelope version
  - compatibility mode
  - serialization timestamp
  - payload

Forward compatibility is handled via ForwardCompatible<T> which preserves
unknown fields through serde(flatten) into a HashMap<String, Value>.
